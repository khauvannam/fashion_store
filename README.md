## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/johndoe/my-awesome-project.git
   cd my-awesome-project
   ```

2. Install npm packages:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Access the application in your browser at:
   ```
   http://localhost:3000
   ```
